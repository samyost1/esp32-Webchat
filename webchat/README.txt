Run de chat app met een terminal in de uitgepakte project folder en de command: docker compose up -d
Dan kan er in de webbrowser een tab met https://localhost geopend worden.
Hier is dan de chat app te zien met je user-id bovenin.
Nog een tab openen met https://localhost en je kan met de chat van het andere tabblad communiceren.

De ports 80 443 1884 worden gebruikt.
